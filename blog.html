Important FAQ

What is the Document Object Model (DOM)?

DOM stands for 'Document Object Model' Document refers to the HTML document. Object Model is the object that represents the HTML document, which Represents the Document as JavaScript Object, maintaining the parent-child relationship between the html elements/tags in a Tree-Like Structure. Includes all necessary Functions/Methods that can CRUD the document and applied CSS styles to that document.

How do you select an element from the DOM?

In JavaScript you can use several methods depending on how specific you want to be in targeting the element. Here are the most common methods: getElementById(), getElementsByClassName(), .getElementsByTagName(), querySelector() ect...

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where instead of adding individual event listeners to each child element, you attach a single event listener to a parent element. When an event occurs on a child element, the event bubbles up to the parent, and the listener on the parent can handle it. You can determine which specific child element triggered the event. Using the event object's .target property.
it is very useful because Cleaner code, Less memory consumption, Simplifies dynamic content handling, Improved performance etc....

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript in the DOM by following these methods like getAttribute() ,setAttribute(), removeAttribute() Add or remove classes using classList.
